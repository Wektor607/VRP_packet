Главынй модуль
================

.. automodule:: main
    :members:
    :undoc-members:
    :show-inheritance:

Главный модуль для работы со всеми алгоритмами оптимизации маршрутов.
    1. В него подаются файлы с данными; 
    2. Есть возможно посмотреть, как просмотреть документацию к эспериментальному стенду;
    3. Можно выбрать алгоритм для решения своей задачи, а настройка параметров под задачу происходит автоматически.
